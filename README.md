Calculate multiplication:
===================================================================================================

Calculate multiplication go inside console from src folder and type  

Multiply-master\src> java -jar multiply.jar --<alg1/alg2> <number1> <number2>  

Algorithms:
===================================================================================================  
Alg1
===================================================================================================  
Alg1 uses my own alg...  
Example:  
  
Input:  
Multiply-master\src> java -jar multiply.jar --alg1 12345678901234567890 11111111111111111111  
  
output:  
137174210013717420998628257899862825790
